[Next.js](https://nextjs.org/) проект, созданный с помощью [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Запуск

Сервер разработки:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Сайт доступен по адресу: [http://localhost:3000](http://localhost:3000).
